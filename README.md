# 《学透：高阶学习者的压缩操作系统》

> **英文：** *Mastery Through Compression: A Unified Framework for Learning*
>
> **副标题：** 理解即压缩——学习方法的统一框架

本书定位：**给有5年以上技术/专业经验的高阶学习者的学习操作系统**，基于「压缩即理解」统一理论，不是技巧的叠加，而是一个可执行的完整管道。

全书共 **四卷 + 序幕 + 终章，15章 + 附录**，目标8-10万字（中文）。
